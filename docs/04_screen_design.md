# 画面設計

## 1. 画面一覧

### 公開
- `/` ホーム
- `/timeline` タイムライン
- `/events` イベント一覧
- `/events/[id]` イベント詳細（ルーム含む）
- `/projects` プロジェクト一覧（検索/タグ/ソート）
- `/projects/[id]` プロジェクト詳細
- `/users/[id]` ユーザページ

### ログイン後
- `/onboarding` フォーカス選択
- `/dashboard` ダッシュボード
- `/messages` 会話一覧
- `/messages/[id]` 会話スレッド
- `/settings` 設定（フォーカス/通知）

### 管理者
- `/admin` イベント管理 / 告知管理

---

## 2. 主要画面UI（要点）

### 2.1 ホーム（/）
- 目的: 「いま何が起きているか」を一目で見せ、投稿を促す
- セクション例（フォーカスで並び替え）
  - タイムライン（最新）
  - 未解決相談ピックアップ
  - 最近更新されたプロジェクト
  - 最近の動き（アクティビティ）
  - イベント
- CTA
  - 「進捗を投稿」
  - 「相談する」
  - 「作品を投稿」

### 2.2 タイムライン（/timeline）
- タブ/フィルタ
  - 全体 / フォロー中 / イベント / プロジェクト
- 投稿カード
  - 投稿者、タイプ（進捗/相談/見せびらかし）
  - 本文（相談はテンプレ表示）
  - リアクション（5種）
  - コメント
  - 相談は「解決済み」「ベスト回答」表示

### 2.3 プロジェクト詳細（/projects/[id]）
- タブ
  - 概要 / 更新 / レビュー / コメント / 仲間募集 / 応援 / 会話
- 応援タブ
  - 外部支援リンク
  - 支援記録作成（支援者）
  - 承認待ち（受領側）
  - 確定支援額（confirmedのみ）
- 会話タブ
  - project会話（グループ）への導線

### 2.4 ダッシュボード（/dashboard）
- 自分のプロジェクト一覧（下書き/公開/アーカイブ）
- 承認待ち支援記録（受領側）
- 未読DM/会話
- 自分の未解決相談
- 投稿CTA（進捗/相談）

### 2.5 設定（/settings）
- フォーカス選択の変更
- 通知設定
  - 種類×チャネル（アプリ内/メール）
  - 将来のダイジェスト（プレースホルダ）

## 3. 導線（例）

### 3.1 継続開発導線
1. 作品を投稿 → 公開
2. 進捗をタイムラインに投稿
3. 相談投稿 → ベスト回答 → 解決済み
4. project会話で仲間と相談
5. 更新投稿 → タイムラインに流れる → 応援が集まる

### 3.2 支援導線（外部投げ銭＋記録）
1. 応援タブで外部リンクへ
2. 支援後に支援記録作成（awaiting_owner）
3. 受領側が「受け取りました」で確定（confirmed）
