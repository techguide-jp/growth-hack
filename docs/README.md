# ハッカソン発・継続開発コミュニティサービス（仕様ドラフト）

このZIPには、以下の成果物が入っています。

- `01_spec.md` : 仕様書ドラフト（機能要件/非機能要件/運用方針/リスクなど）
- `02_api_design.md` : API設計（REST例、SvelteKit実装を想定）
- `03_db_design.md` : DB設計（PostgreSQL想定、主要テーブル定義）
- `04_screen_design.md` : 画面設計（画面一覧/主要UI/導線）
- `05_component_design.md` : コンポーネント設計（SvelteKit + Svelteコンポーネント）
- `06_estimate.tsv` : 見積りシート（TSV）
- `07_questions.tsv` : 質問表（TSV）
- `08_prompt_sveltekit_mock.txt` : モック生成プロンプト（SvelteKit想定、日本語）

## 前提（確定事項）
- フレームワーク: **SvelteKit**
- ユーザロール: **user**（全員同一）と **admin**（運営のみ）
- 投げ銭: **外部決済**。ただし金額はシステム内で「支援記録」として把握
  - JPY固定、期限なし
  - 支援者が記録作成 → 受領側（プロジェクトオーナー）が「受け取りました」で確定
- タイムライン: 投稿タイプ **進捗/相談/見せびらかし**（全公開）
  - 相談テンプレは固定（状況/困りごと/試したこと/環境）
  - リアクションは **5種類**
- メッセージ: **個人DM（1対1）** と **project会話（グループ）**
  - 添付なし（テキストのみ）
- 通知: アプリ内 + メール（即時）
  - 将来: ダイジェスト（日次/週次）を各ユーザが設定で選べる余地
